Created during HiWi position in AI4Grids Project at HTWG Konstanz.

A command line script to simplify ML projects for servers with multiple users. It uses Docker containers to manage and isolate dependencies and tracks experiments with MLFlow. CI is supported with CML runners. Environments can be accessed using jupyter servers. Works with Conda and different CUDA versions.
